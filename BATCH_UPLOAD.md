# 商品批量上传功能说明

## 功能概述
提供Excel批量上传商品的功能，支持一次性上传多个商品信息，提高商品管理效率。

## Excel模板字段说明

| 字段名 | 必填 | 说明 | 示例 |
|--------|------|------|------|
| 商品名称 | 是 | 商品的名称 | 苹果 |
| 分类名称 | 是 | 商品所属分类 | 水果蔬菜 |
| 价格 | 是 | 商品销售价格 | 19.99 |
| 原价 | 否 | 商品原价，用于显示折扣 | 29.99 |
| 库存 | 是 | 商品库存数量 | 100 |
| 状态 | 否 | 商品状态(on/off)，默认on | on |
| 描述 | 否 | 商品详细描述 | 新鲜苹果，产地山东 |
| 图片URL | 否 | 商品图片链接 | https://example.com/apple.jpg |

## 使用步骤

### 1. 下载模板
- 点击"下载模板"按钮获取标准Excel模板
- 按照模板格式填写商品信息

### 2. 填写数据
- 确保必填字段完整
- 价格和库存需为数字格式
- 状态字段使用"on"(上架)或"off"(下架)

### 3. 上传文件
- 点击上传区域或拖拽Excel文件
- 支持.xlsx和.xls格式
- 文件大小不超过10MB

### 4. 查看结果
- 上传完成后显示每条记录的处理结果
- 显示成功和失败的数量统计

## 注意事项

1. **重复商品处理**：如果商品名称已存在，则更新该商品信息
2. **分类处理**：如果分类不存在，系统会自动创建新分类
3. **数据验证**：系统会验证必填字段和数据格式
4. **批量处理**：支持一次上传最多1000个商品

## 错误处理

- 文件格式错误：检查Excel文件是否损坏
- 数据验证失败：查看错误信息并修正数据
- 网络上传失败：检查网络连接后重试
